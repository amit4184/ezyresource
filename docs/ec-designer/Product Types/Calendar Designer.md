## **Introduction**

Calendar Template Creator allows users to easily design custom calendar templates. These templates come with pre-designed date pad layouts, fonts, and color options, which users can personalize by adjusting text styles, colors, and event collections. With an easy-to-use design, this tool makes it simple to create professional-looking calendars without any design experience.
 The calendar templates you create here will be used by customers in the EzyDesigner to customize and order printed calendars.

### Features that EzyDesigner Calendar Template Creator Offers

**Access Pre-designed Templates -** Quickly start designing by using ready-made templates available on the dashboard.

**Add Multilingual Events -** Support for events in various languages for region-specific calendar designs.

**Customize with Design Elements** - Personalize your template using calendar date pads, text fields, images, cliparts, and other design tools for various occasions such as festivals, birthdays, and professional events.

**Use Predefined Placeholders** - Add content easily with predefined placeholders for photos and text.

**Add Backgrounds & Text Colors** - Apply background colors and modify text colors to enhance the visual appeal.

**Edit Unpublished Templates** - Make changes to existing templates before publishing them live.

**2D Design Preview** - View your designed template with a 2D product preview feature before finalizing.

**Advanced Filtering** - Filter templates by occasion, language, published/unpublished status, or template name.

## **Section I: Template Creator Features**

### **Navigation Menus**

* The template name and product name is displayed at the left top corner.

### **Upload Photos**

* Click on the Photos tab to add your own photos. Supported file type is jpg/jpeg.
* PLEASE DO NOT USE CMYK IMAGES,IF  they are uploaded they will be converted to normal JPG and it is in this process something can go wrong with the colors and it also depends on the printer's profile.



### **Photo Holder**

* Photo holders is a frame for holding the photo. These are predefined ratios for the holders, having square,portrait and landscape orientations.
* Image placeholder is a dummy image created in different ratios and orientations for easy and fast designing.
* Photo Holders- Click on Photo Holders, and it shows an image holder creator, Now click on it, and it will create an image holder on the designer. 








### **Background**

* Backgrounds are images that are placed behind the main photos.These are selected backgrounds we currently offer.
* You can add your own backgrounds from the admin section.

  *NOTE - The images need to be jpg format. PLEASE DO NOT USE CMYK IMAGES,IF  they are uploaded they will be converted to normal JPG and it is in this process something can go wrong with the colors.*



### **Clipart**

* Click on Background, and it shows a list of background images. Click on any background image to apply it to the designer.
* Clipart are stickers that can be used to enhance your templates.
* You can add your own cliparts from the admin section. The file formats for cliparts are png and are transparent images.
* Click on clipart, and it will show a list of clipart. Click on it to apply to the designer





### **Text**

* Text: You can add text, but emojis are currently not supported.
* Click on the “Floating text” option to add a text field in the designer.
* Click on text to add a text field on the designer. 
* Note - The default label on the text field ie. 'Enter Your text' will not be rendered if not edited. Any other text will be rendered and printed on the product. Eg. if a text field is added and no editing is done for it. Then it will not get rendered.

  **Text Options:**
*  Template Creator has standard text options like “Locking/Locking” the text box, “Move Forward/Backward” option to move the textbox as desired. “Delete” option to delete the textbox. Font style & size selection. Font alignment option, Bold/Italics option and also the text color/text background color option. 
* The Center part of the designer is a work area.
* The work area will contain an image holder and date pad displayed when the designer is invoked for the first time.
* The red dots on the image are the handles that can be used to resize the image and date pad as required.















#### **Text Options:**

* Lock/Unlock, Move Forward/Backward, Delete
* Font style & size
* Alignment, Bold, Italics
* Text & background color

### **Work Area**

* Contains date pad and image holder by default.
* Resize using red handles.

### **Toolbar for Photo Editing**

* **Lock / Unlock** –
  this feature allows the user to lock/unlock any object in the design area.
* **Duplicate** – 
  allows you to copy a selected photo.
* **Bring Forward** – 
  Photos or text can be arranged one above the other using the Bring Forward button.
* **Send Back** – 
  Works similar to Bring Forward and sends the photo/text holder behind.
* **Crop** – 
  use Crop to edit a photo, you can crop a portion and remove unnecessary parts of the photo.
* **Add Photo** – 
  Allow to add a photo holder.you can select and drop an image on it.
* **Delete** – Deletes selected photo holder

### **Text Editing Tools**

* **Duplicate** – 
   Duplicate allows you to copy a selected photo.
* **Font & Size** – 

  Select the font and size for the text required.Currently we have set a few pre selected fonts.

  Fonts can be enabled from the admin section.
* **Font Color & Background** – 
  Select the text color and background. The current version allows only pre-set colors, the next version will have a color picker with a palette.
* **Alignment** – 
  Text alignment can be done using the above tool.
* **Bold / Italic** – 
  The text can be changed to bold or Italic by toggling these buttons.

### **Save / Preview / Close**

* **Save** – To save the designed template
* **Preview** – 
  Click on preview to view the template on the product, currently only 2D preview is available.
* **Close** – 

  Close will close the template and redirect back to the admin website.

  (always save before closing)

### **Gridlines**

* Gridlines are provided to align two or more photo holders, text fields etc. when the photo/text field is moved in towards the other photo holder/text field objects. The gridlines appear automatically to make it easier to align with other objects.





### **Holiday List**

* **Default (Fixed Holidays):**
  Fixed holidays fall on the same date every year like New Year, Independence Day etc.
* **Non-Fixed Holidays:**
  Non-fixed or movable holidays do not have a set date each year. Examples include: Holi Diwali etc
* Click on default to view all fixed/ default holiday list 





### **Date Style & Design**

* Date Style - It includes all the options to modify the date pad style, such as date pad design, font, and text color
* Date design - A Date Pad Style is the way dates and weekday names are displayed in calendars. You can select your predefined date pad style.
* This feature allows users to dynamically modify the dimensions to fit different designs or user preferences.
* This feature allows users to dynamically modify the dimensions of the date pad by adjusting the width and height according to their requirements.
* *NOTE- The set width and height will be applied to the date pad across all pages*















#### **Typography Options:**

* Typography: You can select your preferred font for the date pad by clicking on the dropdown button and choosing a font style.
* You can adjust the font size by clicking on the “+” and “-” buttons.
* Click on the color picker to display all the available color options. Select any color and click the Apply button. This option allows you to change only the color of the dates on the date pad.
*  Choose all your preferred options and click the apply button. The date pad for all months will be updated according to your selection.









## **Section II: How to Create a Template**

### **Step 1: Login**

* Go to: Admin Page [

  ](https://admin.ezycreate.com)
* Login with valid credentials.

### **Step 2: Access EzyAdmin**

* Click on **Templates** from the left menu.

### **Step 3: Load Existing Templates**

* Click on **My Templates**.

### **Step 4: Create New Template**

* Click on **Create New Group**

#### **Form Fields:**

* **Template Name**: Displayed in EzyDesigner.
* **Products**: Select calendar product.
* **Occasions**: Select from dropdown.
* **Tags**: Add relevant hashtags.
* **Create**: Open in a new tab.
* **Cancel**: Discard.

### **Template Design View**

* One calendar page opens with:

  * Default date pad
  * Image holder
  * Design tools
* **Date Pad Note**:

  * Auto-added for 12 months.
  * Locked to avoid accidental edits.
  * Navigate using > and < arrows.
* **Save / Preview / Close** available on top.

### **Edit an Existing Template**

* Go to the template group.
* Click the **edit icon** to modify.
* Re-publish changes to apply.

### **Edit Template Details**

* Click the edit icon to change name/language.
* Note: Template name and product cannot be changed post-creation.

## **Section III: How to Create Collection and Events**

### **Step a: Calendar Events Dashboard**

* Click on **Calendar Events** under Designer Settings.

### **Step 1: Open Dashboard**

### **Step 2: Create New Collection**

* Click **Create New Collection**.
* Enter:

  * **Collection Name**
  * **Language**
* Click **Save**

### **Step 3: View Collection**

* Listed with name, status, and actions.

### **Step 4: Create Events**

* Click the **calendar icon**.

### **Step 5: Add New Event**

* Click **Create New Event**.
* Add:

  * **Date**
  * **Event Title**
  * **Fixed** = Yes
* Click **Save**

**Note**: Use correct language.
 E.g., Japanese event → 元日 (がんじつ)

### **Event Types**

* **Fixed**: Same date each year (e.g., New Year).
* **Non-Fixed**: Different date each year (e.g., Holi).

### **Step 6: Event List**

* Shows:

  * Date
  * Title
  * Fixed status
* View on **date pad** by clicking the calendar icon.

### **Step 7: Edit or Delete Events**

* Click **Edit** or **Delete**.

### **Step 8: Event View Filter**

* Filter by **Year** and **Month**.

### **Step 9: Color Code**

* **Red Dot**: Fixed Event
* **Green Dot**: Non-Fixed Event

### **Step b: Collection Options**

* Click on **three dots** beside collection name:

  * **Publish**: Publish the collection
  * **Copy**: Duplicate
  * **Delete**: Remove

### **Step c: After Publishing**

* When reopened in Template Creator, events will:

  * Appear in **Holiday List**
  * Show on the **calendar date pad**
